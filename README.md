**Author:** Qasim Ali   

---

## Project Overview

This is a small, responsive multi-page website built using **HTML** and **CSS**. The project demonstrates frontend skills, including page layout, navigation, responsive design, and form creation.  

The website contains four pages:  

- **Home** – Welcome page with a brief introduction.  
- **Services** – Lists the services offered.  
- **About Us** – Provides information about the company.  
- **Contact** – Includes a contact form with Name, Email, and Message fields.  

The navigation bar is consistent across all pages and the layout is responsive, adapting to laptop and tablet screens.

---

## Features

- **Responsive Design:** Works on desktop, tablet, and laptop using CSS media queries.  
- **Consistent Navigation:** Same navigation bar across all pages.  
- **Contact Form:** Includes HTML validation for required fields.  
- **Clean Layout:** Simple, user-friendly design focusing on readability.
